# AoC2024

Welcome to the Advent of Code[^aoc] Kotlin project created by [mullertom][github] using the [Advent of Code Kotlin Template][template] delivered by JetBrains.

This repository provides solutions for the puzzles using [Kotlin][kotlin] language.
